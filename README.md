## 🚀 **Quick Start**

### **Prerequisites**
- Node.js 18+
- Python 3.8+
- MySQL 8.0+

### **Setup**
```bash
# Backend setup
cd Backend
pip install -r requirements.txt
# Configure .env file with database and email settings
python app.py

# Frontend setup
cd Frontend
npm install
npm run dev
```

### **Access**
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
